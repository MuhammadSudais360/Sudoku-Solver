# Sudoku Solver in C++

This project solves a Sudoku puzzle using the Backtracking Algorithm.

## Features

- 2D Array Representation
- Backtracking Technique
- Sudoku Rule Checking
- Recursive Solution

## How to Run

### Compile

```bash
g++ main.cpp -o sudoku
```

### Run

```bash
./sudoku
```

## Concepts Used

- Arrays
- Functions
- Recursion
- Backtracking Algorithm
